  <summary>インデックス(Index)</summary>
  ある列(カラム)の要素について、ある規則性に沿って並び替えられたもの。
  
  インデックスがないと、線形探索となりo(n)となるが、整列されていることで計算量を削減し
  効果的なパフォーマンス向上が得られる。
  
  ただしデメリットとして、データ追加時にその都度インデックス作成するため、データ追加の処理が重たくなる。
  
  よって無闇にインデックスを使うのでなく、効果的に選択する必要がある。
